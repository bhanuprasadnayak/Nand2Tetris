# Nand2Tetris: Building a Modern Computer from First Principles

This repository contains my solutions and implementations for the projects from the **Nand2Tetris** course, based on the book *The Elements of Computing Systems* by Noam Nisan and Shimon Schocken.

The course provides a hands-on approach to understanding how a modern computer works by building an entire computing system from the ground up — starting from basic logic gates and progressing to an operating system and high-level applications.

## Projects Completed

* **Project 01:** Boolean Logic
* **Project 02:** Boolean Arithmetic
* **Project 03:** Sequential Logic
* **Project 04:** Machine Language
* **Project 05:** Computer Architecture
* **Project 06:** Assembler
* **Project 07:** VM I – Stack Arithmetic
* **Project 08:** VM II – Program Control
* **Project 09:** High-Level Language (Jack Program)
* **Project 10:** Compiler I – Syntax Analysis
* **Project 11:** Compiler II – Code Generation
* **Project 12:** Operating System

## About Nand2Tetris

Nand2Tetris is an educational project that guides learners through constructing a complete computer system from elementary logic gates to a functional software hierarchy. It offers a comprehensive understanding of the interaction between hardware and software components.
